Bonjour !

Pour ouvrir et utiliser l'application merci de suivre les étapes suivantes:



0. Si python3 n'est pas déjà installé sur votre ordinateur :
   Rendez-vous sur https://www.python.org/downloads/ pour télécharger la version convenant à votre système d'exploitation
   Ouvrez le fichier téléchargé et poursuivez l'installation

1. Ouvrez le terminal et rendez vous à l'intérieur du dossier nommé application

2. Installez les librairies nécessaires au bon fonctionnement du programme
	Pour cela tapez dans le terminal:
	pip install pysqlite3
	pip install pandas

3. Executez ensuite simplement le programme en tapant dans le terminal
	python netflop.py

4. Vous pouvez désormais parcourir l'application et la base de données



Si vous rencontrez un problème avec l'une de ces étapes merci d'installer et d'utiliser Thonny 
en suivant ces étapes suivantes:

1. Rendez-vous sur https://thonny.org/ pour télécharger la version convenant à votre système d'exploitation.
   Ouvrez le fichier téléchargé et poursuivez l'installation.
   
2. Une fois dans le logiciel,
   Installez les librairies nécessaires au bon fonctionnement du programme
	Pour cela allez dans Tools
	Puis Manage Packages...
	Dans la nouvelle fenêtre qui viens de s'ouvrir recherchez sqlite3 et pandas et installez les

3. Ouvrez et executez le fichier netflop.py

4. Vous pouvez désormais parcourir l'application et la base de données
